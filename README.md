# Water-Indicator-Arduino

This is the Setup of Water-Level-Indicator using an arduino board.
This Setup contains:
-Arduino Uno board
-5 Jumper Wires.
-Water level indicator.
-A Buzzer.
-A Cup of Water.
-A laptop or pc to power the arduino board.
